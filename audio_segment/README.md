## Audio Segment Tool

Segment the Videos and Audios files into small segments of audios.

## Requirements

- [FFmpeg][ffmpeg]
- pydub: `pip install pydub`

## Support formats

* Input format: `mkv`, `mp4`, `flv`, `wav` or other formats that FFmpeg surpported
* Output format: `wav`

Refer to `test_audio_segment.py`.
